# entregable1
## Descripción
El programa analiza un conjunto de números enteros, comprobando si el valor máximo se repite, calculando la media y evaluando el resultado.
## Como funciona
Cálculo del valor máximo del array; Comprobación de repetición del valor máximo; Cálculo de la media aritmética y Evaluación de la media (BIEN / MAL)
